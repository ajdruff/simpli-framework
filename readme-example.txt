The Simpli Hello World plugin is a demonstration plugin that can be used as a template to create new WordPress plugins using the Simpli WordPress Plugin Framework. Simpli is a Simple Plugin development framework that helps you create WordPress Plugin quickly within a fully object oriented structure.

== Installation ==
1. Upload the 'simpli-hello' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to the Settings page in the admin sidebar in the dashboard.
4. etc.
